# Upbit Bollinger Band Alert Bot (Telegram)

업비트 시세 데이터를 기반으로 **볼린저밴드 하단선 하향 돌파 시 텔레그램 알림**을 보내고,  
텔레그램에서 **`/now` 명령어로 현재가 조회**가 가능한 파이썬 봇입니다.

- 업비트 **공개 API(시세 조회)**만 사용하므로 **Access/Secret Key 불필요**
- 텔레그램 Bot API를 사용하여 알림/응답 메시지 전송

---

## 주요 기능

### 1) 볼린저밴드 하단선 하향 돌파 알림
- 지정한 티커(기본 `KRW-BTC`)의 OHLCV를 가져와 볼린저밴드를 계산합니다.
- 현재가가 **하단선을 위→아래로 교차(cross-down)** 하는 순간을 감지하면 텔레그램으로 알림을 전송합니다.
- 동일 조건에서 알림이 과도하게 반복되지 않도록 **쿨다운(cooldown)**을 적용합니다.

### 2) 텔레그램 `/now` 명령어 지원
- 텔레그램 채팅에서 `/now`를 입력하면 봇이 **현재가**를 답장합니다.
- 그룹 채팅에서는 텔레그램 설정(privacy mode)에 따라 `/now@봇아이디` 형태가 필요할 수 있습니다.

### 3) 실행 시작 알림
- 프로그램을 실행하면 봇이 시작되었다는 메시지를 텔레그램으로 1회 전송합니다.

---

## 동작 방식 요약

- `refresh_sec`(기본 5초)마다 다음을 수행합니다:
  1. 텔레그램 업데이트(getUpdates)를 확인하여 `/now` 명령을 처리
  2. 업비트 OHLCV를 다시 가져와 볼린저밴드를 갱신
  3. 업비트 현재가를 조회하여 하단선 하향 돌파 여부 판단
  4. 돌파 감지 시 텔레그램 알림 전송(쿨다운 고려)
